Website Kedai Kopi

Proyek ini merupakan website pemesanan untuk kedai kopi dengan integrasi API Midtrans. Pelanggan dapat melihat daftar menu, memilih produk yang diinginkan, dan melakukan pemesanan secara online. Sistem pembayaran sudah terintegrasi dengan Midtrans sehingga transaksi dapat dilakukan secara aman, cepat, dan praktis.
